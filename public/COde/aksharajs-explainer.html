8 Engines in 1

Everything AksharaJS does for you

♿

Accessibility Engine

Automatically fixes your website so everyone can use it — including people who use screen readers, keyboards only, or have visual impairments.

WCAG 2.1 AA ADA Compliant ARIA Labels Focus Indicators
🌐

Language Engine

Automatically detects Telugu, Hindi, Arabic, Tamil and 10+ other scripts. Sets the right direction (RTL/LTR) and correct language code without any configuration.

Telugu Devanagari Arabic RTL Auto-detect
🔍

SEO Engine

Injects page titles, meta descriptions, Open Graph tags, Twitter cards, and Google-readable structured data — all derived from your existing content.

Open Graph JSON-LD Meta Tags Canonical URL
📴

Offline / PWA Engine

Makes your website work even without internet. Registers a Service Worker that caches your pages and shows a fallback when the user goes offline.

Service Worker Cache First Offline Page PWA Manifest
🔒

Security Engine

Follows OWASP best practices. Adds Content Security Policy, secures all external links, blocks XSS attacks, and warns if you're not on HTTPS.

OWASP Top 10 XSS Prevention CSP Headers Secure Links
🗣️

Voice / TTS Engine

Lets users hear your content read aloud. Automatically detects the language and speaks it in the right accent — including Indian language voices.

Web Speech API Auto Language Click to Speak
📷

OCR Engine

Reads text from images — including images written in Telugu, Hindi, or Tamil. Useful for accessibility: screen readers can then speak the image content.

Tesseract.js Indic Scripts Auto Alt Text
💾

Offline Storage

Saves data inside the user's browser so your app works offline. A simple, clean API — no databases, no servers, no complicated setup needed.

IndexedDB Promise-based Auto-init
How It Works

Three steps. That's it.

From zero to fully enhanced in under a minute.

1

Install the package

One command in your terminal: npm install aksharajs. No complicated setup, no peer dependencies, no configuration files needed.

2

Import & initialise

Add two lines to your JavaScript file. Tell AksharaJS which features you want — accessibility, SEO, language detection, security, etc.

3

Call enhance(document)

AksharaJS scans your entire page and silently fixes every problem it finds. Your visitors never see it working — they just experience a better website.
