# YuktAI

AI Engine

---

## 🚀 Overview

YuktAI is an open-source AI engine for JavaScript that enables developers to run intelligent logic locally using a plugin-based runtime.

---

## ✨ Features

- ⚡ AI Runtime Engine
- 🔌 Plugin-based architecture
- 🌐 Works in Node.js and browser
- 🔒 No external APIs required
